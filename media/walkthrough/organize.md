# Projects, renames and guardrails

- **Projects** — group repositories (frontend, backend, e2e…) into one named project with combined totals and a single PDF receipt/invoice.
- **Rename** — give hash-named remote repositories like `(unknown) 2bebdc79` a friendly display name (✎ button in the repository detail).
- **Hide** — remove noise repositories from the dashboard (🙈 button); raw exports still include them.
- **Star** — pin your important repositories to the top.
- **Session cost alert** — set `copilotCostLens.sessionCostAlertUsd` and get warned the moment a single (agent) session crosses your threshold.
